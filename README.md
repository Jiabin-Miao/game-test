# 可供选择的游戏类别
## 角色扮演游戏 (RPG)
**简介**：
- 玩家扮演角色，通过任务、战斗提升能力，体验故事与成长
- 强调剧情沉浸、角色养成、世界观探索，可融合动作、策略等元素
- 在宏大叙事中掌控角色命运

**目录路径**：`.\test-game\template\RPG`

## 动作游戏 (ACT)
**简介**：
- 考验实时操作与反应，完成高难度动作挑战
- 快节奏战斗、精准操控、流畅动作设计，常见于平台跳跃、格斗等场景
- 极致手眼协调与爽快感

**目录路径**：`.\test-game\template\ACT`

## 策略游戏 (SLG)
**简介**：
- 通过规划与资源管理达成目标，注重逻辑与决策
- 战术布局、兵力调配、长期规划，可分为即时战略（RTS）或回合制（TBS）
- 运筹帷幄的智力对抗

**目录路径**：`.\test-game\template\SLG`

## 格斗游戏 (FTG)
**简介**：
- 一对一对抗，通过招式连击与精准时机击败对手
- 帧级操作、角色特性克制、华丽连段，强调竞技性与技巧深度
- 纯粹反应与策略的较量

**目录路径**：`.\test-game\template\FTG`

## 第一人称射击游戏 (FPS)
**简介**：
- 以玩家视角操控武器射击，强调沉浸式战斗
- 真实射击体验、快速反应、战术配合，常见于军事或科幻题材
- 紧张激烈的多人对抗

**目录路径**：`.\test-game\template\FPS`

## 第三人称射击游戏 (TPS)
**简介**：
- 以第三方视角控制角色射击，兼顾动作与战术
- 更灵活的动作（如闪避）、战术观察，融合动作或生存元素
- 战术机动与策略战斗

**目录路径**：`.\test-game\template\TPS`

## 冒险游戏 (AVG)
**简介**：
- 以解谜与探索推动剧情，弱化动作或战斗
- 叙事驱动、环境互动、逻辑谜题，风格多样（如悬疑、奇幻）
- 沉浸于故事与世界观

**目录路径**：`.\test-game\template\AVG`

## 模拟游戏 (SIM)
**简介**：
- 模拟现实或虚构系统的运作，追求真实或创造性体验
- 高度还原规则（如驾驶、经营），或开放创造（如沙盒）
- 掌控虚拟世界的规则

**目录路径**：`.\test-game\template\SIM`

## 多人在线战术竞技游戏 (MOBA)
**简介**：
- 团队对抗，摧毁对方核心建筑（如“水晶”）
- 5v5分工协作、英雄角色养成、实时战术调整
- 高强度团队竞技与策略

**目录路径**：`.\test-game\template\MOBA`

## 大型多人在线角色扮演游戏 (MMORPG)
**简介**：
- 大规模玩家在线互动，共享开放世界与长期成长
- 社交驱动、公会系统、经济体系，结合PvE/PvP玩法
- 在虚拟社区中体验史诗冒险

**目录路径**：`.\test-game\template\MMORPG`

## 动作角色扮演游戏 (ARPG)
**简介**：
- 融合动作与角色扮演，实时战斗与角色养成并重
- 流畅动作操作、技能树成长、装备系统，弱化传统RPG的回合制
- 兼具打击感与深度角色发展

**目录路径**：`.\test-game\template\ARPG`

## 策略角色扮演游戏 (SRPG)
**简介**：
- 回合制战斗与战术策略结合，角色成长与职业系统为核心
- 网格地图、角色定位（如坦克/治疗）、战场策略，强调团队搭配
- 精密战术与角色培养
 
**目录路径**：`.\test-game\template\SRPG`

## 竞速游戏 (RAC)
**简介**：
- 驾驶载具竞速，追求速度与技巧
- 真实物理或夸张操控，赛道设计多样（如公路、越野）
- 肾上腺素飙升的驾驶快感

**目录路径**：`.\test-game\template\RAC`

## 体育游戏 (SPG)
**简介**：
- 模拟真实体育规则，还原运动竞技体验
- 精准操作、战术模拟、真实球员/球队授权，支持多人对抗
- 代入职业运动员视角

**目录路径**：`.\test-game\template\SPG`

## 益智游戏 (PZL)
**简介**：
- 通过逻辑、空间或反应解谜，达成关卡目标
- 规则简洁、挑战思维、轻松上手，风格从休闲到硬核
- 脑力激荡的成就感

**目录路径**：`.\test-game\template\PZL`
